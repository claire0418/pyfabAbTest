How to Record Video

Using the GUI

Navigate to the video tab.

Click the Record button to start recording. This will open a video file with the specified file name, overwriting any existing video with that name. Frames will be written to the file until recording is stopped. The number of frames in the video is presented on the LCD display.

Click the Stop button to stop recording.

Select the file name by clicking on it. This will launch a file selection dialog.

Using keyboard shortcuts

  • Type "r" to start recording.
  • Type "r" again or "s" to stop recording.
  • Type "R" to launch the file selection dialog.
