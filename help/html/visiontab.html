Vision Tab

The vision tab provides capability for end-to-end holographic tracking and characterization both in real-time and after a video is recorded.

After recording video, data can be exported as pylorenzmie Videos, containing Frames and Trajectories. Frames contain the Features found in each frame, and Trajectories contain Features representing the same feature in as many frames that it appears.

If either the Estimate or Refine pipelines are chosen, then the z(t) vs. t Trajectory plot and a_p vs n_p probability distribution are populated with data from the video.
