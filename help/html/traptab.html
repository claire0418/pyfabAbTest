Trap Tab

The trap tab presents real-time information on the optical traps in the current trapping pattern. For each trap, the interface presents the following information:

x, y, z
Current position of the trap in image coordinates [pixel].
alpha
The amplitude of the trap relative to other traps in the pattern (Default: 1).
phi
The phase of the trap relative to the incident beam [radians]. (Default: random)

Entries for traps appear when traps are created and disappear when traps are destroyed. Changing any value changes the associated property of the particular trap in real time. The graphical representation of the trap also updates to reflect the change.
