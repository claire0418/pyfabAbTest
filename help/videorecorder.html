Video Recorder

The video recorder panel acts as a digital video recorder.

Clicking the Record button creates a video file whose name is listed in the file name box, and starts recording video frames into that file. An existing file will be overwritten.

Clicking the Stop button stops recording and closes the video file.

The frame counter presents the current number of frames in the video file.

Clicking on the file name box launches a file selection dialog that can be used to change the name of the video file.

Keyboard shortcuts

Typing "r" starts recording. Typing "s" stops recording. Typing "R" brings up the file selection dialog.
